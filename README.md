## About This Project
This is my database project for the final exam that I did with my team. Unfortunately, this project can only be executed properly if your display resolution is 1920 x 1080 and your scale is 100%. You can change it in the display settings. C# Marketplace Program that allow user to Buy and Sell mini-market products. There are 4 roles in this program customer, rider, admin, and seller. This program is fully integrated with database for CRUD.
